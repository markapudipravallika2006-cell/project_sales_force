# AI-Powered Job Recruitment & Candidate Management System

## Phase 1: Requirement Analysis

### Introduction
The AI-Powered Job Recruitment & Candidate Management System is a Salesforce-based application designed to simplify the recruitment process.

### Problem Statement
- Duplicate candidate records
- Lost applications
- Delayed interviews
- Manual hiring process

### Business Requirements
- Candidate Management
- Resume Upload
- Interview Scheduling
- AI Resume Screening

### Project Scope
The project focuses on managing the recruitment process using Salesforce CRM.

### Objectives
- Automate recruitment
- Improve hiring efficiency
- Reduce manual work
